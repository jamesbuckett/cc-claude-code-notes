# Rule #1 - Prompt Skills not Claude 
* Most work is REPETATIVE TASKS
* Capture those taks as SKILLS
  * Use Plugin `skill-creator`
* Skills are organized collections of files that package composable procedural knowledge for agents
  * Just Folders
  * A way to get a Tesk done

## Personal Skill Ideas
* Build a Educational Website to learn the theory of a topic
* Build a Educational Tutorial Website to learn the practical of a topic
* Prepare for Weekly One on One
* Layers
*   Skills = Applications
*   Agents = Operating System = Today Prompts
*   Models = CPU Processors 

```
Prompt
Based on recent sessions, what tasks am i doing repeatedly that should be a skill instead of one-off prompts?
For each one, suggest a skill name and what context it would need.
```
